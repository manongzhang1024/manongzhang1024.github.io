Typora配置文件上传到github

1.修改偏好设定

点击文件-偏好设置

按照下图配置图片设置，点击下载或更新设置上传服务

2.创建github仓库

创建一个仓库设置Public，这里我设置的仓库路径是cloudingimg

3.获取git token

点击头像-settings-最下面的developer settings-Personal access tokens，然后点击generator new token创建新token

勾选repo复选框，将失效时间设置一下，这里我设置的无过期时间

保存一下生成的token

4.修改Typora配置文件

进入偏好设置

修改仓库路径和你的token

可以点击验证图片上传选项验证一下是否上传成功
